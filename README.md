# Multi-Role User Management System

## 📌 Overview
This project implements a role-based access control system using React and Spring Boot.

## 🚀 Features
- User role management (Admin, User, etc.)
- CRUD operations
- Role-based access control
- Secure backend APIs

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- API: REST

## 📂 Project Structure
- frontend/ → React application
- backend/ → Spring Boot application

## ▶️ How to Run
### Backend
- Open backend folder
- Run Spring Boot application

### Frontend
```bash
cd frontend
npm install
npm start
